# Safety mode

+ Safety mode is only available on NVIDIA Drive platforms (QNX).

+ This example is runnable on all platforms:
	+ On supported platforms, it builds and loads a safety engine.
	+ On unsupported platforms, it exits with `[SKIP]` messages.

+ Steps to run.

```bash
python3 main.py
```
