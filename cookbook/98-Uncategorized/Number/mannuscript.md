| 概念        | 本质                | 技术细节                                                                | 使用场景                   |
| ----------- | ------------------- | ----------------------------------------------------------------------- | -------------------------- |
| **TF32**    | **计算精度格式**    | 19-bit（1-8-10），Ampere Tensor Core 内部使用，存储时仍用 FP32          | NVIDIA GPU 矩阵运算加速    |
| **NVFP8**   | **NVIDIA 量化策略** | 使用 `float8_e5m2` + `float8_e4m3fn` + 自定义缩放，非新数据类型         | NVIDIA H100/Blackwell 推理 |
| **MXFP8**   | **OCP 标准规范**    | 元素用 `float8_e5m2`/`float8_e4m3fn`，block size 32，scale 用 `E8M0`    | 通用 AI 训练/推理          |
| **MXFP4**   | **OCP 标准规范**    | 元素用 `FP4 E2M1`，block size 32，scale 用 `E8M0`                       | 极限压缩场景               |
| **NVFP4**   | **NVIDIA 变体规范** | 元素用 `FP4 E2M1`，block size 16，scale 用 `E4M3` + `FP32` tensor scale | Blackwell GPU 推理         |
| **FP4e0m3** | **理论格式**        | 1-0-3，等同于 INT4，无实用价值                                          | ❎ 无实际应用               |
