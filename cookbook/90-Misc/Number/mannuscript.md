| Concept     | Nature                           | Technical Details                                                               | Use Cases                       |
| ----------- | -------------------------------- | ------------------------------------------------------------------------------- | ------------------------------- |
| **TF32**    | **compute precision format**     | 19-bit (1-8-10), used internally by Ampere Tensor Cores, still stored as FP32   | NVIDIA GPU matrix acceleration  |
| **NVFP8**   | **NVIDIA quantization strategy** | uses `float8_e5m2` + `float8_e4m3fn` + custom scaling; not a new data type      | NVIDIA H100/Blackwell inference |
| **MXFP8**   | **OCP standard specification**   | elements use `float8_e5m2`/`float8_e4m3fn`, block size 32, scale uses `E8M0`    | general AI training/inference   |
| **MXFP4**   | **OCP standard specification**   | elements use `FP4 E2M1`, block size 32, scale uses `E8M0`                       | extreme compression scenarios   |
| **NVFP4**   | **NVIDIA variant specification** | elements use `FP4 E2M1`, block size 16, scale uses `E4M3` + `FP32` tensor scale | Blackwell GPU inference         |
| **FP4e0m3** | **theoretical format**           | 1-0-3, equivalent to INT4, no practical value                                   | ❎ no practical application      |
