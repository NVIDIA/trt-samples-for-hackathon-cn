
usage: nsys sessions list [<options>]

	-f, --output-format=<value> [default: plain]

	   Controls the output format.
	   Possible values are: ['json', 'plain'].
	   Export format type.

	-h, --help=[<tag>]

	   Print the command's help menu. The switch can take one optional
	   argument that will be used as a tag. If a tag is provided, only options
	   relevant to the tag will be printed.
	   The available help menu tags for this command are:

	   head, header, and list.

	-p, --show-header=<false|true> [default: true]

	   Controls whether a header should appear in the output.
